# SQL Sales Analysis Project

## 📊 Project Overview
This project focuses on extracting, transforming, and analyzing sales data to uncover business insights. Using SQL, I explored customer demographics, product performance, and revenue trends over time.

## 📁 Project Structure
- **Data/**: Contains the raw datasets in `.csv` format (`dim_customers`, `dim_products`, `fact_sales`).
- **SQL_Scripts/**: A collection of SQL queries used for data cleaning and analysis.

## 🛠️ Skills Demonstrated
- **Data Cleaning:** Handling NULL values in `order_date` and ensuring data integrity.
- **Aggregations:** Using `GROUP BY` and `DATETRUNC` to summarize sales by year and month.
- **Data Modeling:** Working with a Star Schema (Fact and Dimension tables).

## 🚀 How to Use
1. Clone the repository.
2. Import the `.csv` files into your SQL database (SQL Server, PostgreSQL, etc.).
3. Run the scripts in the `SQL_Scripts/` folder to generate the analysis.

## 📈 Key Queries Included
- Yearly Sales Growth
- Customer Purchase Patterns
- Product Category Analysis
